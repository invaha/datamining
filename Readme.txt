# پیش‌بینی ریسک دیابت با استفاده از یادگیری ماشین

این پروژه به بررسی و مقایسه‌ی روش‌های مختلف یادگیری ماشین برای پیش‌بینی ریسک ابتلا به دیابت می‌پردازد. هدف، انتخاب بهترین مدل برای دسته‌بندی افراد در معرض خطر و همچنین شناسایی گروه‌های مختلف افراد بر اساس ویژگی‌های آن‌ها با استفاده از خوشه‌بندی است.

## کتابخانه‌های مورد نیاز

برای اجرای این پروژه، کتابخانه‌های زیر مورد نیاز است:

*   `pandas`: برای کار با داده‌ها و ساخت دیتافریم.
*   `matplotlib`: برای رسم نمودارها و تجسم داده‌ها.
*   `scikit-learn`: برای پیاده‌سازی الگوریتم‌های یادگیری ماشین و ارزیابی مدل‌ها.
*   `scipy`: برای خوشه‌بندی سلسله مراتبی.

برای نصب این کتابخانه‌ها می‌توانید از pip استفاده کنید:

```bash
pip install pandas matplotlib scikit-learn scipy
